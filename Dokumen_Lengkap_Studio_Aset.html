Alur Kerja & Panduan Pengguna Studio Aset

Terbitan: 25 Februari 2026 | Versi Sistem 2.1

I. Alur Kerja (Workflow) Sistem

Sistem manajemen ini dirancang untuk memastikan transparansi antara pendataan di kantor pusat (Admin) dan audit nyata di lapangan (Operator).

Persiapan Oleh Admin

Manajemen Struktur

Admin mengatur "Digital Twin" dari studio fisik. Ini meliputi pendaftaran gedung, ruangan, hingga katalog barang (Master Asset). Tanpa input data dari Admin, Operator tidak memiliki daftar untuk dicek.

Eksekusi Oleh Operator

Audit Lapangan

Operator melakukan validasi harian. Mereka membandingkan data di layar ponsel dengan barang fisik di ruangan. Hasil pengecekan ini akan mengupdate kondisi aset secara otomatis di dashboard pusat.

üîÑ Analisis Sinkronisasi Real-Time

Aplikasi menggunakan teknologi sinkronisasi otomatis. Setiap hasil checklist yang dikirim Operator akan:

  • Mengubah status aset di database pusat saat itu juga.
  • Menciptakan Log Aktivitas yang mencatat histori perpindahan barang.
  • Memberikan notifikasi pada Admin jika ada aset yang dilaporkan "Rusak" atau "Mati".

II. Panduan Penggunaan: ADMIN

Gunakan langkah-langkah berikut untuk mengelola infrastruktur aset studio:

1

Definisikan Lokasi & Ruangan

Masuk ke dashboard Admin, pilih menu Locations untuk menambah gedung, lalu Rooms untuk menambah studio/ruangan di setiap gedung.

2

Registrasi Master Asset

Daftarkan semua inventaris baru di menu Master Assets. Masukkan kategori yang tepat (misal: Lighting, Sound, PC) agar laporan mudah difilter.

3

Plotting Aset ke Ruangan

Gunakan menu Asset Room untuk memindahkan barang dari gudang/katalog ke ruang spesifik. Tahap ini wajib agar barang muncul di daftar cek Operator.

4

Monitoring Laporan Audit

Pantau menu Audit Reports untuk meninjau hasil kerja Operator dan menu Logs untuk melihat riwayat aktivitas sistem.

III. Panduan Penggunaan: OPERATOR

Gunakan langkah-langkah berikut saat melakukan pengecekan di studio:

1

Pilih Target Ruangan

Buka aplikasi, klik Mulai Checklist, lalu pilih Gedung dan Ruangan studio yang akan diperiksa kondisinya.

2

Verifikasi Barang

Cocokkan setiap item di aplikasi dengan barang fisik. Pilih status BAIK, RUSAK, MATI, atau HILANG.

3

Menu Perpindahan (Movement)

Jika menemukan barang ruangan lain di ruangan ini, gunakan fitur Pindahkan ke Sini agar lokasi aset terupdate secara resmi di sistem.

4

Catatan & Pengiriman

Berikan catatan tambahan mengenai kebersihan atau kendala ruangan lainnya, lalu klik Submit untuk mengirim laporan audit Anda.
