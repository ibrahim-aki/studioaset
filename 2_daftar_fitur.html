Katalog Fitur Aplikasi Studio Aset

Dokumen ini menjabarkan kemampuan struktural dan fitur rinci dari sistem aplikasi.

UMUM Modul Autentikasi & Keamanan

  • Login System: Otentikasi berlapis berdasarkan peran kredensial dengan rute navigasi terproteksi otomatis.
  • Enkripsi Otomatis: Sesi pengguna dijaga oleh manajemen state Next.js dan Firestore backend.
  • Pembaruan Akun Wajib: Fitur wajib ganti password (Change Password Modal) yang mencegat pengguna di layar mana pun jika terdeteksi bahwa password tersebut membutuhkan pembaruan.

ADMIN Modul Manajemen Struktural

  • Dashboard Analytics: Menyajikan panel informasi holistik tentang jumlah aset, laporan hari ini, dan rangkuman aktivitas.
  • Manajemen Lokasi Cabang: Mendata lokasi-lokasi bisnis (CRUD - Create, Read, Update, Delete).
  • Manajemen Ruangan (Rooms): Organisasi tata letak ruang per area cabang. Admin dapat mengatur ruang untuk diinspeksi.
  • Manajemen Master Assets: Inventarisasi alat-alat studio yang dipetakan pada setiap ruang (misal: Kamera, Lensa, Lampu).
  • Investigasi Audit Log (Logs Track): Panel pengawasan jejak digital real-time tak terhapuskan. Admin bisa memfilter kejadian berdasarkan: Sistem (aset/cabang baru), Perpindahan (alat digeser antaruang), Status (alat rusak), maupun Akses (jam login/logout). Sangat krusial untuk investigasi kehilangan barang.
  • Laporan & Inspeksi Harian (Checklists View): Gudang data hasil inspeksi Operator. Dilengkapi peringatan otomatis jika ada laporan "BARU". Admin dapat melihat jam pasti inspeksi, mengecek kondisi (Baik/Rusak/Mati/Hilang), membaca keluhan catatan, mengunduh foto bukti asli dari lapangan, serta memantau letak perpindahan terakhir alat tersebut.
  • Pemantauan Status Live Ruangan: Admin dapat melihat plang status Live Now, Ready For Live, Stadby, atau Not Ready pada masing-masing studio dari jarak jauh berdasarkan submission laporan terakhir.
  • Akses Sistem Changelog: Halaman transparan untuk melihat riwayat commit pembaruan perangkat lunak, bug fixes, dan penambahan fitur langsung dari Repository Pusat, lengkap dengan konversi zona waktu lokal.

OPERATOR Modul Inspeksi & Laporan Lapangan

  • Blokir Desktop & Validasi Perangkat: Sistem secara proaktif memblokir akses pengguna jika dicurigai berasal dari Personal Computer (Desktop). Hanya ponsel/tablet yang diizinkan untuk operasional cek aset.
  • Sensor Kamera Paksa: Validasi ketat perangkat keras pengguna. Fitur tidak akan mengizinkan pekerja memulai tugas jika tidak ada perangkat kamera atau izin kamera dimatikan.
  • Modul Shift Kerja: Tombol interaktif untuk "Mulai Shift" dan "Selesai Shift" sebagai indikator operasional real-time.
  • Akses Kamera Terintegrasi (In-App Camera): Pengambilan foto barang dilakukan pure di dalam browser tanpa perlu membuka aplikasi kamera bawaan HP, menghindari manipulasi galeri.
  • Daftar Penilaian (Checklists Form): Daftar aset siap di-scan mata oleh operator, dilapisi dengan form status cepat (Baik, Rusak, atau Perlu Perawatan) dan keterangan catatan.
  • Sinkronisasi Laporan Instan: Data inspeksi diformat lengkap dengan waktu dan pelaksana, lalu otomatis terlempar ke database untuk langsung bisa diawasi Admin.
